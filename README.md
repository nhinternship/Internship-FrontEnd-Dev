<h2 align=center> 📝 How to Contribute? </h2>  

Interns can contribute by:<ul><li>Implementing small, well-defined features or components (e.g., a new button, a form validation logic, a carousel).</li><li>Fixing reported bugs or issues that are scoped appropriately for a beginner.</li><li>Writing unit and integration tests for existing or new code.</li><li>Refactoring small sections of code to improve readability or performance.</li></ul>
